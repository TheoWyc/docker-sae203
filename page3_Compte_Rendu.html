Compte rendu de choix des services adaptés



Pour notre application nous avons choisi de faire un site en html et css.

Nous avons aussi choisi du javascript pour faire les lecteurs mp3, l’ajout et le téléchargement des musiques.



Nous avons ensuite créé une image docker partant d’un debian sur laquelle nous avons installé apache2 pour héberger le site et php pour faire une base de données contenant les comptes des utilisateurs et leurs statuts (premium ou gratuit) que nous n’avons pas eu le temps d’installer.



Puis nous avons mis le projet sur github avec sur la branche principale le fichier dockerfile contenant les instructions de l’image docker et nous avons aussi ajouté le fichier README.md.



Nous avons créé ensuite une branche gh-pages qui contient le site qui est donc hébergé sur github.
